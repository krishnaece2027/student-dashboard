# student-dashboard
Flask-based student management dashboard with CRUD operations


## Features
- Add student
- View all students
- Update student details
- Delete student

## Tech Stack
- Python
- Flask
- SQLite
- HTML (Bootstrap)

## How to Run
1. Install Flask
2. Run app.py
3. Open http://127.0.0.1:5000/
